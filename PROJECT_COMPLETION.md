## 🚀 How to Get Started

### **STEP 1: Local Testing (5 minutes)**

```bash
cd "d:\Redberyl Work\First Task\app"
pip install -r requirements.txt
python app.py
```

Open http://localhost:5000 in your browser ✅

### **STEP 2: Docker Build (10 minutes)**

```bash
cd "d:\Redberyl Work\First Task\app"
docker build -t cicd-app:latest .
docker run -d -p 5000:5000 --name cicd-app-container cicd-app:latest
curl http://localhost:5000
```

Verify container is running ✅

### **STEP 3: Docker Hub Push (10 minutes)**

```bash
docker login
docker tag cicd-app:latest YOUR_USERNAME/cicd-app:latest
docker push YOUR_USERNAME/cicd-app:latest
```

Image available on Docker Hub ✅

### **STEP 4: GitHub Setup (10 minutes)**

```bash
cd "d:\Redberyl Work\First Task\app"
git init
git add .
git commit -m "Initial CI/CD project"
git remote add origin https://github.com/YOUR_USERNAME/cicd-app.git
git push -u origin main
```

Code on GitHub ✅

### **STEP 5: Jenkins Pipeline (15 minutes)**

1. Create Jenkins credentials
2. Create pipeline job pointing to Jenkinsfile
3. Click "Build Now"
4. Watch 7 stages complete ✅

## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Docker daemon not running | Start Docker Desktop |
| Python dependencies missing | `pip install -r requirements.txt` |
| Port 5000 in use | Close other apps on that port |
| Image won't push to Docker Hub | Run `docker login` first |
| Container won't start | Check logs with `docker logs` |
| Jenkins can't find files | Verify "Script Path" is `app/Jenkinsfile` |

**Full troubleshooting:** See SETUP_GUIDE.md
